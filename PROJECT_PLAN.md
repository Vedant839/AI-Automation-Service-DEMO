# Project Plan

Based on the mock contract effective 15 June 2026.

## Phase 1: Discovery & Workflow Analysis

Contract duration: 7 days

Template tasks:

- Map student support journeys
- Collect course, batch, scholarship, fee, and FAQ documents
- Review CRM data structure and API feasibility
- Identify sensitive fields that should not be sent to AI providers
- Define escalation rules for counselors and parents
- Confirm dashboard recipients and report timing

Outputs:

- Completed discovery questionnaire
- Integration checklist
- Data privacy risk notes
- Client-specific configuration draft

## Phase 2: MVP Development

Contract duration: 30 days

Template tasks:

- Configure AI support assistant knowledge sources
- Implement FAQ and scheduling response workflow
- Configure engagement triggers
- Generate initial teacher intelligence report
- Add admin usage instructions
- Run internal testing with sample and staging data

Outputs:

- Chatbot workflow
- Engagement automation workflow
- Teacher report workflow
- Test cases and known limitations

## Phase 3: Deployment & Monitoring

Contract duration: 8 days

Template tasks:

- Deploy approved workflows to production channels
- Set up monitoring and alert review
- Onboard staff
- Validate response quality and trigger reliability
- Document bug-fix and change-request process

Outputs:

- Deployment notes
- Admin instructions
- Acceptance checklist
- Maintenance backlog

## Few-Day Template Build Roadmap

Day 1:

- Finalize reusable docs, prompt templates, config schema, and local demo skeleton

Day 2:

- Add realistic mock workflows, sample test cases, and dashboard/report examples

Day 3:

- Package personalization checklist and prepare a client handoff version

Day 4+:

- Personalize for the real client once APIs, content, CRM fields, and communication channels are confirmed
