# Integration Checklist

## Required for MVP

- CRM API access
- Course documentation
- FAQ documents
- Batch and schedule data
- Attendance data
- Test score data
- Staging environment
- Website chat or embed access
- WhatsApp provider credentials if WhatsApp is included in MVP

## Access Controls

- Use named accounts where possible
- Use least-privilege API keys
- Store secrets outside source control
- Rotate credentials after project completion if required
- Revoke unused credentials after termination

## API Review

- Authentication method confirmed
- Rate limits documented
- Webhook support confirmed
- Error handling expectations confirmed
- Sandbox and production URLs separated
- Data retention rules documented

## Launch Readiness

- Approved FAQ content uploaded
- Escalation contacts configured
- WhatsApp templates approved
- Teacher report recipients approved
- Monitoring owner assigned
- Rollback process documented
