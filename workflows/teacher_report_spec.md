# Teacher Intelligence Report Specification

## Purpose

Give teachers a daily evening summary of learning gaps and student risk indicators.

## Delivery

- Default frequency: daily evening
- Default time: 18:00 local time
- Default channel: email

## Report Sections

- Weak chapters/topics
- Frequently asked doubts
- At-risk students
- Engagement analytics
- Recommended next actions

## Data Inputs

- Student attendance
- Test scores
- Doubt/chat topics
- Course and batch metadata
- Engagement alerts

## Privacy Defaults

- Avoid unnecessary personal data
- Use student IDs instead of names unless approved
- Exclude payment, password, government ID, and banking data
- Share reports only with approved teacher/admin recipients
