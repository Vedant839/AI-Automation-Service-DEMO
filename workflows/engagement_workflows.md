# Student Engagement Automation Workflows

## Purpose

Identify students who may need intervention and notify the right human team before disengagement becomes permanent.

## Trigger Types

- Inactive student
- Low-performing student
- Missed classes
- Test score decline
- Parent notification
- Counselor escalation

## Example Trigger Rules

| Trigger | Default Rule | Action |
| --- | --- | --- |
| Inactive student | No activity for 7 days | Send reminder and alert counselor |
| Missed classes | 2 or more missed classes in 14 days | Notify parent and counselor |
| Low score | Latest score below 50% | Counselor follow-up |
| Score decline | Drop of 15 percentage points | Teacher review and counselor alert |

## Personalization Points

- CRM fields
- Attendance source
- Test score source
- Parent notification policy
- Counselor assignment logic
- WhatsApp template wording
- Escalation priority levels
