# AI Support Assistant Workflow

## Purpose

Reduce routine student support workload while giving students fast, safe answers for course, schedule, fee, scholarship, and common doubt queries.

## Channels

- Website chat widget
- Mobile application chat
- WhatsApp integration

## Standard Flow

1. Student sends a message.
2. System checks for sensitive or unsupported requests.
3. System retrieves matching approved FAQ/course content.
4. AI assistant drafts a response using approved context only.
5. If confidence is low or the topic is sensitive, the assistant escalates to a counselor.
6. Conversation summary is logged according to the client privacy policy.

## Escalation Conditions

- Payment records, passwords, government IDs, banking details, or account security issues
- Student distress, complaint, refund dispute, or policy exception
- Low answer confidence
- Repeated unresolved question
- Parent or counselor requested explicitly

## Personalization Points

- Client FAQ documents
- Course catalog
- Batch schedule source
- Scholarship policy
- Fee policy
- Supported languages
- Handoff team and working hours
